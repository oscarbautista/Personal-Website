Some of the projects that I've worked on

Beat Blocks

Designed to encourage a more collaborative and intuitive music-making process. The Beat Blocks table allows up to four people to mix and match different sounds to create music based on the blocks they place in the middle.

EzNIC

Created to help users with minimum network knowledge to switch the network interface card and modify its IP address, subnet mask, default gateway, and DNS address. Won the Diversified "NIC of Time" challenge at Hack GSU.

Click Clack Crawler

A web page designed to help people who are new to building keyboards find the parts they need to get started. Because most websites don't have all the parts needed to build a keyboard, this is meant to be a resource to help you find the parts you need at the lowest cost.

Frogify

Using the Spotify API, this mobile webpage allows users to create an interactive playlist where anyone at a social gathering can vote for the next played song. Created as a part of a team for UGA Hacks

T2 Cinemas Website

A fully functional movie theater website capable of customer registration, ticket ordering, and displaying movie information. This was implemented in a team of four for Software Engineering class.
